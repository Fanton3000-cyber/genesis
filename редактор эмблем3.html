Управление:
• ЛКМ — добавить вершину
• ПКМ — удалить последнюю вершину
• Двойной клик — замкнуть текущий путь
• Ctrl+Z — отменить последнюю вершину
• N — новый путь (не замыкая текущий)
• Delete — удалить выбранный путь
• Первый замкнутый путь = основа, остальные = вырезы
